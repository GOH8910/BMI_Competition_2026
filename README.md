# BMI Competition

## Git Workflow

### Fetch Latest Changes

```bash
git fetch
git pull
```

### Make Changes

1. Create a new branch:

```bash
git checkout -b "your-branch-name"
```

2. Make your changes, then stage and commit:

```bash
git add .
git commit -m "describe your changes"
```

3. Push your branch:

```bash
git push -u origin "your-branch-name"
```
